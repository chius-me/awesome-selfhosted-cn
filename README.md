# Awesome Self-Hosted CN

[![Awesome](https://awesome.re/badge.svg)](https://awesome.re)
[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg?style=flat-square)](http://makeapullrequest.com)

一个专为中国 HomeLabber 整理的自托管项目精选列表。

与原版 [awesome-selfhosted](https://github.com/awesome-selfhosted/awesome-selfhosted) 不同，本项目并非其中文译版，而是优先收录：
- 🇨🇳 国内开发者主导的优秀项目
- 🔥 在国内网络环境下极其流行、易用的项目
- 🌐 拥有高质量中文本地化支持的项目

## 操作系统
- [fnOS](https://www.fnnas.com/) - 国产 NAS 系统，类群晖 UI，易用性强。
- [ImmortalWrt](https://github.com/immortalwrt/immortalwrt) - OpenWrt 的稳定分支，扩展包更丰富。
- [iStoreOS](https://github.com/istoreos/istoreos) - 基于 OpenWrt 的轻量 NAS/路由系统，对新手友好。
- [OpenMediaVault](https://github.com/openmediavault/openmediavault) - 基于 Debian 的 NAS 系统，插件生态丰富。
- [OpenWrt](https://github.com/openwrt/openwrt) - 广泛应用于路由器的嵌入式 Linux 系统。
- [Proxmox VE](https://github.com/proxmox) - 开源虚拟化平台，整合 KVM 和 LXC，支持 Web 管理。
- [TrueNAS](https://github.com/truenas) - 基于 ZFS 的企业级 NAS/存储操作系统。
- [Unraid](https://github.com/unraid) - 灵活的 NAS 系统，支持 Docker 和 KVM 虚拟化。

## 媒体影音
- [AutoBangumi](https://github.com/EstrellaXD/Auto_Bangumi) - 全自动追番工具，配合 RSS 实现新番自动下载管理。
- [bili-sync](https://github.com/amtoaer/bili-sync) - B 站视频/番剧同步工具，自动下载追番更新。
- [BililiveRecorder](https://github.com/BililiveRecorder/BililiveRecorder) - B 站直播自动录制工具。
- [Immich](https://github.com/immich-app/immich) - 自托管的照片/视频备份与管理平台，功能对标 Google Photos。
- [Jellyfin](https://github.com/jellyfin/jellyfin) - 免费软件媒体系统，Emby 的分支，无任何高级功能限制。
- [Komga](https://github.com/gotson/komga) - 漫画/电子书媒体服务器，支持多用户阅读进度同步。
- [MetaTube](https://github.com/metatube-community/jellyfin-plugin-metatube) - Jellyfin 元数据刮削插件，优化中文影视资源匹配。
- [MeTube](https://github.com/alexta69/metube) - YouTube/视频网站下载的 Web UI 工具。
- [MoviePilot](https://github.com/jxxghp/MoviePilot) - 基于 NASTool 重构的基于 PT 站的自动化观影应用。
- [Navidrome](https://github.com/navidrome/navidrome) - 自托管的音乐流媒体服务，兼容 Subsonic API。
- [PhotoPrism](https://github.com/photoprism/photoprism) - 基于 AI 的照片管理应用，自动标签和分类。

## 网络
- [AdGuard Home](https://github.com/AdguardTeam/AdGuardHome) - 全网广告拦截与 DNS 服务器，保护整个家庭网络。
- [Anubis](https://github.com/TecharoHQ/anubis) - HTTP 请求权重验证防火墙，阻止 AI 爬虫。
- [Cloudflared](https://github.com/cloudflare/cloudflared) - Cloudflare Tunnel 客户端，通过 Cloudflare 安全暴露内网服务。
- [DDNS-Go](https://github.com/jeessy2/ddns-go) - 简单好用的 DDNS 客户端，支持多种域名服务商。
- [frp](https://github.com/fatedier/frp) - 专注于内网穿透的高性能反向代理应用。
- [MetaCubeXD](https://github.com/MetaCubeX/metacubexd) - Mihomo/Clash-Meta 的现代化 Web 面板。
- [Mihomo](https://github.com/MetaCubeX/mihomo) - Clash-Meta 的后继者，基于规则的代理内核。
- [NetBird](https://github.com/netbirdio) - 基于 WireGuard 的零信任网络连接平台。
- [NPS](https://github.com/ehang-io/nps) - 轻量级、高性能、功能强大的内网穿透代理服务器。
- [OpenClash](https://github.com/vernesong/OpenClash) - OpenWrt 上使用的 Clash 客户端，规则管理强大。
- [Tailscale](https://github.com/tailscale) - 基于 WireGuard 的零配置 VPN 组网工具。
- [ZeroTier](https://github.com/zerotier) - 软件定义网络（SDN），轻松组建跨地域虚拟局域网。
